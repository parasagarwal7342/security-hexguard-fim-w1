# Contributing to HexGuard

Thank you for your interest in HexGuard! As a weekly security release, we welcome community feedback and bug reports.

## 🛠️ Development Setup
1. Fork the repository.
2. Install testing dependencies: `pip install pytest`.
3. Ensure all tests pass before submitting a PR: `pytest tests/`.

## 🛡️ Security Policy
If you find a security vulnerability, please do NOT create a public issue. Instead, contact the repository owner directly as this is a security-focused tool.

## 🤝 Contribution Guidelines
* Maintain 100% test coverage for any new features.
* Follow PEP8 coding standards.
* Document any new CLI arguments clearly in the `README.md`.
